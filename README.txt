# Cozy Town Builder

Готовая HTML5-игра в жанре простого уютного градостроителя.

## Что внутри
- `index.html` — основная страница игры
- `styles.css` — оформление
- `app.js` — вся игровая логика

## Как открыть
### Самый простой способ
1. Открой папку проекта в VS Code
2. Открой `index.html`
3. Запусти через расширение **Live Server**

### Можно и так
Просто открыть `index.html` в браузере двойным кликом.

## Что уже реализовано
- карта 7x7
- часть клеток закрыта, можно открывать за монеты
- 3 типа зданий: Дом, Магазин, Парк
- уровни зданий до 3
- пассивный доход раз в секунду
- население, уют и доход в секунду
- бонус соседства от парков
- улучшение зданий
- снос зданий с возвратом части цены
- сохранение в `localStorage`
- адаптация под desktop и mobile

## Под Яндекс Игры
Игра сделана без сервера и без бэкенда. Это обычный HTML5-проект.
Для публикации обычно нужен архив со статическими файлами:
- `index.html`
- `styles.css`
- `app.js`
